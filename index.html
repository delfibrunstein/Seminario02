TP — Algoritmos de ordenamiento · Entrega final · Hace 1 día

# Ordenamiento burbuja
def bubble_sort(lista):
  n = len(lista)
  for i in range(n):
    for j in range(0, n-i-1):
      if lista[j] > lista[j+1]:
        lista[j], lista[j+1] = lista[j+1], lista[j]
  return lista

Explicación: El algoritmo de ordenamiento burbuja es un método de clasificación simple que funciona comparando repetidamente elementos adyacentes e intercambiándolos si están en el orden incorrecto, con una complejidad temporal de O(n²) en el peor caso.

El código funciona bien para listas pequeñas. Es el algoritmo más eficiente para ordenar datos porque hace comparaciones de a dos elementos.
